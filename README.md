# Portfolio Manager

This project is a portfolio manager application.

## Installation

Use [Poetry](https://python-poetry.org/) to install dependencies:

\`\`\`bash
poetry install
\`\`\`

## Usage

Run the application:

\`\`\`bash
poetry run python src/main.py
\`\`\`
